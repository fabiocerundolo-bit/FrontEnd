# FrontEnd

Progetto didattico con esercizi di JavaScript, HTML e CSS suddivisi per lezioni.

## Struttura cartelle

- `Esercizio_0/` - esercizi base e prime prove di interazione.
- `Esercizio03/` - esercizio 3.
- `EsercizioStack/` - loto e pagine multiple.
- `Eserzio_To_Do_List/` - mini progetto ToDo list con pagine.
- `JAVASCRIPT/` - raccolta di script e appunti 2.
- `Lezione_0/` - appunti e esempi iniziali.
- `Lezione_1/` - appunti e script base.
- `Lezione_2/` - progetti di esercizio, incluso Tailwind.
- `Lezione_3/` - esempi con HTML/CSS/JS.
- `Lezione_04/` - switch, pagine e esercizi.
- `Lezione_7/` - callback, form, script principali.

## Come usare

1. Apri `index.html` o il file della lezione che vuoi testare.
2. Lancia un server locale (es. `python3 -m http.server 8000`).
3. Apri `http://127.0.0.1:8000/Lezione_7/callback.html` per il test callback.

## Note specifiche

- `Lezione_7/index.html` ha un semplice form d'esempio.
- `Lezione_7/js/script.js` contiene funzioni di saluto e presentazione corso.
- `Lezione_7/js/callback.js` mostra callback, timer e gestione eventi.

## Miglioramenti possibili

- Aggiungere un file `favicon.ico` in root per evitare 404 nei browser.
- Aggiungere commenti e istruzioni per altri esercizi.
- Convertire tutti i file in UTF-8 se non già.
