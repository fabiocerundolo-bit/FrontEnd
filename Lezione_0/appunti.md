# Cos'è HTML (HyperText Markup Language)

L'**HTML** è un linguaggio di markup utilizzato per definire la **struttura** dei contenuti all'interno di una pagina web.

---

### Come funziona il Markup?
I "markup" (etichette) hanno il compito di **racchiudere** e **avvolgere** il contenuto per istruire il browser su come deve essere visualizzato o interpretato. 

### Esempio Pratico
Se abbiamo la stringa `"Ciao Mondo!"`, possiamo avvolgerla in un tag di **paragrafo** per definirla come testo standard:

```html
<p>Ciao Mondo!</p>
